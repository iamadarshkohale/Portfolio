Networking and Storage

VPN, Network Access Translation,Internet Gateway, S3 lifecycle policy

AWS VPN enables secure connections between your on-premises network and your AWS VPC over the internet. It uses encrypted IPsec tunnels to protect data in transit. VPN is commonly used for hybrid cloud setups, ensuring private and secure communication.

AWS NAT Gateway allows instances in a private subnet to access the internet or other AWS services while preventing inbound traffic from the internet. It acts as a bridge for secure outbound-only communication. NAT Gateways are managed, scalable, and highly available within an Availability Zone.

An Internet Gateway (IGW) is a horizontally scaled, redundant AWS component that enables communication between instances in your VPC and the internet. It allows public subnets to send and receive traffic over the internet. IGW is essential for hosting public-facing applications like websites on EC2
